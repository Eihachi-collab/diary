☁ Google Drive 同期設定
Client ID
Client ID の取得手順：
1. Google Cloud Console でプロジェクトを作成
2. 「APIとサービス」→「ライブラリ」→ Google Drive API を有効化
3. 「認証情報」→「認証情報を作成」→「OAuth 2.0 クライアント ID」
4. アプリの種類：ウェブアプリケーション
5. 「承認済みの JavaScript 生成元」に を追加
6. 作成されたClient IDをここに貼り付けて「ログイン」
同期の仕組み：日記データをGoogle Driveの「マイドライブ」に diary_backup.json として保存します。保存するたびに自動でアップロードされ、起動時に自動でダウンロード・マージされます。
🤖 AIコメント設定
API キー
AnthropicのAPIキーを設定すると「AIからひと言」機能が使えます。
APIキーは console.anthropic.com から取得できます。
キーはこのデバイスのみに保存され、外部には送信されません。
